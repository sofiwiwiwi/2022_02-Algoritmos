La tarea se realizó en python, específicamente la versión 3.9

Para simplificar la entrada, se colocaron los casos de prueba entregados en la carpeta "casos_prueba", así
que para ejecutar el código escribir el siguiente comando en la consola:
    python3 puentes.py < casos_prueba/«archivo.txt»

Donde archivo.txt es el nombre del archivo con el input. Por ejemplo:
    python3 puentes.py < casos_prueba/input3.txt

El output se realizará por consola.
